# Age Calculator API
An API takes in a request body as below

Endpoint: /user

Method: POST

>Sample payload/request:
```JSON
{
   "name": "Rehema",
    "surname": "Jordan",
    "dateOfBirth": "1992-07-07"
}
```

>Output:
```JSON
{
    "name": "Rehema",
    "surname": "Jordan",
    "currentAge": 29,
    "dateOfBirth": null
}
```
